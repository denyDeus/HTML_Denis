# Tic Tac Toe

A modern, responsive Tic Tac Toe game built with **HTML**, **CSS**, and **Vanilla JavaScript**. This project was developed as part of **The Odin Project** JavaScript curriculum to practice modular JavaScript, Factory Functions, the Module Pattern (IIFE), and DOM manipulation.

## 🎮 How to Play

1. Enter both player names.
2. Click **Start Game**.
3. Players take turns placing **X** and **O**.
4. The first player to align three marks horizontally, vertically, or diagonally wins.
5. If all squares are filled without a winner, the game ends in a tie.
6. Click **Restart** to play again.
