# Library App

A simple JavaScript library application that lets users:

- add new books,
- display books on the page,
- mark books as read or unread,
- and remove books from the library.

## Project Files

- index.html - Main page structure
- style.css - Styling for the app
- Book.js - Book constructor and toggle method
- Library.js - Library logic, form handling, rendering, and actions

## Features

- Stores books in an array
- Gives each book a unique id using `crypto.randomUUID()`
- Displays books as cards
- Includes a form for adding new books
- Supports removing books and toggling read status

## How to Run

Open index.html in your browser to use the app.
